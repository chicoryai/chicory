### Available APIs

Chicory provides comprehensive API access for:

- **Projects** - Manage organizations and workspaces
- **Integrations** - Connect data sources, code repositories, and tools
- **Scan** - Manage context scanning and preparation
- **Agents** - Create, configure, and deploy AI agents
- **Tasks** - Manage agent tasks
- **Tools** - Manage MCP tools
- **Evaluation** - Manage evaluations for an agent
- **MCP Gateway** - Manage MCP tools and gateway configurations
- **Health & Monitoring** - Track performance and system status
- *(and more as the platform evolves)*
