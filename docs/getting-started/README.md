# How to Get Started

To begin leveraging Chicory AI, the first step is to connect your primary data assets. This allows the Data Understanding Module to start building the initial version of your "living context". 

* [**Quick Start Guide**](building-your-first-agent/README.md) - Build and deploy your first AI agent in minutes
* [**Building Your First Agent Cookbook**](building-your-first-agent/docs/1-agent-creation.md) - Complete guide covering the full Agent Development Life Cycle (ADLC) from creation to production deployment
* [**Comprehensive Learning:**](../how-to-guides/README.md) - Step-by-step guides, tutorials, and ready-made recipes for solving real-world data engineering and AI agent use-cases.

These hands-on experiences are the best way to understand Chicory AI's practical capabilities and how it can be applied to your specific data challenges.
