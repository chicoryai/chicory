# Required Secrets

Add these in your GitHub repository **Settings → Secrets and variables → Actions**:

- `CHICORY_API_TOKEN` → API token from Chicory
- `CHICORY_AGENT_ID` → Agent ID from Chicory
- `GCP_PROJECT_ID`  → Project ID from the terraform service account secret generation in Bigquery
- `GCP_SERVICE_ACCOUNT_KEY`  → Service Account Key from the terraform service account generation in Bigquery
- `JIRA_API_TOKEN`  → JIRA API token creation : https://id.atlassian.com/manage-profile/security/api-tokens
- `JIRA_BASE_URL` → JIRA Base URL
- `JIRA_EMAIL` → User JIRA Email ID
