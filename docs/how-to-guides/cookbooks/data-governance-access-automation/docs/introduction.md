# Introduction

This cookbook shows how to use **Chicory AI agents** that can be used to help automate data access requests in an enterprise 

### Tools & Integrations
- **BigQuery** – SQL execution and runtime insights
- **REST API** – Deployment mechanism for Chicory agents

---
**Problem**: Manual approval processes for data warehouse access requests create delays that block analysts and data consumers from completing time-sensitive work. Approval workflows lack context about requester's roles and data sensitivity, leading to slow turnaround times.

### Quick Start

1. Set up JIRA ticket creation and automation rules 
2. Build the agent and deploy it 
3. Set up GITHUB action workflows
4. Watch the PR generation, analyze the agent response + terraform code generation
5. Approve and merge the PR/ Deny and close the PR
6. Watch the JIRA ticket getting updated with the status
